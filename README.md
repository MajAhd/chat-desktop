# Electron Desktop Chat

Electron.js , React.js , Socket.io

![alt text](./public/img/Screen.png)


## Available Scripts:

`npm install`

`npm run start`

`npm run build`

## Technologies

- Electronjs
- reactjs
- redux
- Socket.io

## How It Work

- User Auth With Phone Number & Email
- Unique User Name
- Peer To Peer Chat
- Attach Photo && Movies File
- Contacts
